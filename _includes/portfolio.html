Devign Hunt

My final project was to build a meet-up style website for freelance designers and developers to find other freelancers to work together on their individual projects and to share their ideas for projects.

The project was built on a Mean stack using AngularJS, Node.js, Express and MongoDB. Utilizing Passport for local authentication, running the back-end API on Heroku and hosting the front-end on Github pages.
